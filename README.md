# Barber Shop Website

## Overview

This website that shows a barbershop services, work, and contact info. It allows the customer to reach the social media of all working barbers, reserve appointments, see the works, the reviews of customers. with smoothly animation in each section. It also includes the location of the barbershop.

## Features

Desktop/tablet/mobile responsive design
Navigation menu
Animated sections using GSAP
Scroll animations
Work gallery slider
Reviews slider
Highlighting of the active navigation link
Scroll to top button
Stunning UI

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* GSAP Animation Library

## Website Sections

* Home
* About
* Services
* Work Gallery
* Testimonials
* Contact Information

## Purpose

This project was created to enhance the marketing of a barbershop exsists in Egypt.
